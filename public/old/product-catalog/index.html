Validating your Product Catalog

For the best results, your product catalog should include a range of products covering at least 3 different primary categories.
Items shown with a red cross are urgent and should be fixed before continuing.
  1. Description of icon when needed

    No missing fields

    File contains all required fields for product details and all required dimensions.

  2. Description of icon when needed

    # Product Rows

    Loader has identified that you have provided more than the minimum number of required products.

  3. Description of icon when needed

    # Categories

    Loader has identified that you have provided more than the minimum number of required category options.

  4. Description of icon when needed

    Unexpected Product Catalog fields

    Uploaded catalog must contain only the 22 fields that appear in the product template file (nothing past column V).

    See Help Guide
  5. Description of icon when needed

    Missing Product Catalog fields

    Uploaded catalog must contain all of the required product catalog fields: '_id', 'name', 'url', 'imageUrl', 'price', 'listPrice', 'rating' & 'numRatings'.

    See Help Guide
  6. Description of icon when needed

    Missing Dimension fields

    Uploaded catalog must contain all of dimension fields required for the Affinity Wheel: 'category', 'itemClass', 'style' & 'brand'.

    See Help Guide
  7. Description of icon when needed

    Insufficient Number of Products

    Loader has identified that your catalog does NOT contain the minimum 15 required products. You can continue to upload the data but you should consider adding more products and categories for better personalisation results.

    See Help Guide
  8. Description of icon when needed

    Insufficient Number of Categories

    Loader has identified that your catalog does NOT contain the minimum 3 required category options. You can continue to upload the data but you should consider adding products from more categories for better personalisation results.

    See Help Guide
