How To Get Your Account

If you're using Interaction Studio directly via the URL, and NOT in your MC LA, then the Account will be the subdomain of the URL. E.g. if the URL is https://interactionstudio.evergage.com, then you would enter interactionstudio in the Account field.

If you're using Interaction Studio via your MC LA, then you can find your Account by opening up Chrome DevTools -> Network Tab, then go to Interaction Studio in your MC LA. As the page loads, click on ui/ in the left hand side, and the Request URL will contain the actual URL being used for Interaction Studio. You then enter the sub-domain of the URL. E.g. if the URL is https://ccharalambous1477350.australia-3.evergage.com then you would enter ccharalambous1477350.australia-3 in the Account field.

Issues With "View Time" in Actions Builder  

Any "View Time" on a product MUST be paired with a "View Item" on the same product, otherwise Interaction Studio won't show anything for the "View Time" on its own.

Insufficient Number of Categories  

To build realistic customer profiles the tool tries to group users into a preferred category interest. If not enough categories are provided it results in all the created profiles being too similiar and this can lead to a poor demo experience. Try to upload at least 3 categories all with five products or more.

Insufficient Number of Products  

The tool is designed to create a different customer profiles as possible. If not enough products are provided it results in all the recipes being too similiar and this can lead to a poor demo experience. Try to upload at least 3 categories all with five products or more.

Missing Dimension or Product Catalog fields  

This error results because you have edited the field structure of the template file. The structure should not be changed and the following fields all need to be populated:

  • _id (string) - The unique identifier of the item such as a product id or SKU
  • name (string) - Name of the product
  • url (string) - Link to this item
  • imageUrl (string) - Fully qualified url for the image representing this item
  • price (decimal) - Current price of the item e.g. 9.99
  • listPrice (decimal) - Alternative list price e.g. MSRP for comparison or to illustrate discount e.g. 11.99
  • rating (decimal) - The average rating score for reviews on this item e.g. 4.5
  • numRatings (number) - The count of reviews on this item
  • category - Tag to be applied to the Category Dimension
  • itemClass - Tag(s) to be applied to the Item Class Dimension (use ';' for multiple values)
  • style - Tag(s) to be applied to the Style Dimension (use ';' for multiple values)
  • brand - Tag(s) to be applied to the Brand Class Dimension (use ';' for multiple values)

For more details see the Product Catalog Guide on Solutions Central
